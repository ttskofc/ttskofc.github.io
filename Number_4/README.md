# React + Ant Design — Переписанный сайт

Это минимальный шаблон проекта на Vite + React + Ant Design,
в который перенесена ваша предыдущая HTML+CSS страница.

Что в архиве:
- package.json — зависимости и скрипты
- index.html — корневой HTML
- src/main.jsx — точка входа
- src/App.jsx — основной компонент (Layout + Menu + Card)
- src/index.css — минимальные стили (mobile-first)
- инструкции по установке

## Как запустить (локально)

1. Распакуйте архив.
2. Откройте папку проекта в терминале.
3. Выполните:
   ```
   npm install
   npm run dev
   ```
4. Откройте URL, который покажет Vite (обычно http://localhost:5173).

## Пояснения — что я сделал и почему

1. **Vite + React** — лёгкий современный старт. Vite обеспечивает быстрый dev-server.
2. **Ant Design** — используется для быстрых, аккуратных компонентов (Layout, Menu, Card).
3. **Mobile-first** — CSS написан так, чтобы сначала задавать поведение для мобильных,
   затем через media query менять вид для десктопа.
4. **Order через CSS** — порядок блоков (таблица → ссылки → форма) управляется стилями,
   как ты просил — HTML-структура при этом осталась понятной.
5. **Заглушки** — в блоках "Список гиперссылок" и "Форма" оставлены placeholder'ы,
   чтобы ты мог заполнить их позже.
